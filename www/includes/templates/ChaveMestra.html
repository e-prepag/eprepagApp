O que é uma chave mestra?

A chave mestra é um recurso de segurança superimportante para proteger suas informações sensíveis. É um código único e seguro que permite acesso controlado a dados confidenciais, como os do seu sistema de Ponto de Venda (PDV).

Como usar a chave mestra:

Quando você fizer uma compra, o sistema vai pedir para validar sua identidade. Para isso, basta inserir a chave mestra que está neste e-mail.

Se você fizer uma compra em outro dispositivo ou usar dispositivos diferentes dentro de 7 dias, o sistema vai pedir a chave mestra novamente.

Em caso de dúvidas, entre em contato com suporte@e-prepag.com.br ou com o Chat no nosso site oficial: https://www.e-prepag.com.br/.
