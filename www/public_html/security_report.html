Exemplos Práticos de Correção - SQL Injection

Exemplo 1: Webhook de Pagamento

Código Vulnerável:

$idVenda = $_POST['id_venda'];
$sql = "SELECT * FROM vendas WHERE vg_id = " . $idVenda;
$informacaoVenda = SQLexecuteQuery($sql);

Código Corrigido:

$stmt = $pdo->prepare("SELECT * FROM vendas WHERE vg_id = ?");
$stmt->execute([$_POST['id_venda']]);
$informacaoVenda = $stmt->fetch();

Exemplo 2: Sistema Administrativo

Código Vulnerável:

$operadora = $_GET['operadora'];
$sql = "select opr_nome, opr_codigo from operadoras 
        where opr_status = '" . $operadora_ativada . "' 
        and opr_codigo <> " . $opr_teste;
$resopr = SQLexecuteQuery($sql);

Código Corrigido:

$stmt = $pdo->prepare("SELECT opr_nome, opr_codigo 
                       FROM operadoras 
                       WHERE opr_status = ? 
                       AND opr_codigo <> ?");
$stmt->execute([$operadora_ativada, $opr_teste]);
$resopr = $stmt->fetchAll();

Exemplo 3: Módulo de Estatísticas

Código Vulnerável:

$data_inicio = $_POST['data_inicio'];
$data_fim = $_POST['data_fim'];
$sql = "SELECT * FROM vendas_estado 
        WHERE data_venda BETWEEN '$data_inicio' AND '$data_fim'
        AND estado = '" . $_POST['estado'] . "'";
$vendas_estado = SQLexecuteQuery($sql);

Código Corrigido:

$stmt = $pdo->prepare("SELECT * FROM vendas_estado 
                       WHERE data_venda BETWEEN ? AND ?
                       AND estado = ?");
$stmt->execute([
    $data_inicio,
    $data_fim,
    $_POST['estado']
]);
$vendas_estado = $stmt->fetchAll();

Recomendações Gerais para Prevenção de SQL Injection

  • Sempre use prepared statements com parâmetros vinculados
  • Nunca concatene variáveis diretamente em strings SQL
  • Valide e sanitize todas as entradas de usuário
  • Use tipos de dados apropriados para os parâmetros
  • Implemente um sistema de logs para queries SQL
  • Use um ORM (Object-Relational Mapping) quando possível

Implementação Segura da Função SQLexecuteQuery

Nova Versão Segura:

class Database {
    private $pdo;

    public function __construct() {
        $this->pdo = new PDO(
            "mysql:host=localhost;dbname=seu_banco",
            "usuario",
            "senha",
            [PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]
        );
    }

    public function executeQuery($sql, $params = []) {
        try {
            $stmt = $this->pdo->prepare($sql);
            $stmt->execute($params);
            return $stmt->fetchAll(PDO::FETCH_ASSOC);
        } catch (PDOException $e) {
            $this->logError($e);
            throw new Exception("Erro na execução da query");
        }
    }
